1. load 메서드
    - 사용자가 지정한 URL 주소에 데이터를 전송하고 외부 콘텐츠를 요청하여 가져올 때 사용

    $("요소 선택").load(url, data, 콜백 함수)
        - url : 외부 콘텐츠를 요청할 외부 주소를 입력
        - data : 전송할 데이터를 입력
        - 콜백 함수 : 전송이 완료되면 콜백 함수에 지정한 코드가 실행


2. $.ajax() 메서드

    $.ajax({
        url : 전송 페이지,
        type : 전송 방식,
        data : 전송할 데이터,
        dataType : 요청한 데이터 형식,
        success : fuction(data){
            전송에 성공하면 실행될 코드;
        },
        error: function(){
            전송에 실패하면 실행될 코드;
        }
    })
